# Foundations

**Duration**: 2-3 weeks | **Difficulty**: Beginner to Intermediate

This section covers the fundamental building blocks of DSA. Master these before moving forward.

## Learning Order

1. **Big O Notation** (2-3 days)
   - Understanding time and space complexity
   - Essential for analyzing any algorithm

2. **Arrays** (3-4 days)
   - Most fundamental data structure
   - Learn common manipulation patterns

3. **Strings** (3-4 days)
   - Similar to arrays but with specific techniques
   - Heavy in interviews

4. **Hash Tables** (3-4 days)
   - Critical for optimization
   - Enables O(1) lookups

## Key Techniques

After learning the data structures, master these patterns:

- **Two Pointers**: Efficient array/string traversal
- **Sliding Window**: Subarray/substring problems
- **Frequency Counting**: Using hash maps to count occurrences
