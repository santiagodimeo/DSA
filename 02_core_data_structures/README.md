# Core Data Structures

**Duration**: 3-4 weeks | **Difficulty**: Intermediate

Master these essential data structures that appear in most technical interviews.

## Learning Order

1. **Linked Lists** (4-5 days)
   - Singly and doubly linked lists
   - Reversal, cycle detection, merge

2. **Stacks & Queues** (3-4 days)
   - LIFO and FIFO operations
   - Monotonic stack pattern
   - Implementation using arrays and linked lists

3. **Trees** (5-6 days)
   - Binary trees, BST
   - Traversals (inorder, preorder, postorder, level-order)
   - Recursion and iteration

4. **Heaps** (3-4 days)
   - Min and max heaps
   - Priority queues
   - Top K problems

## Key Techniques

- **Fast & Slow Pointers**: Cycle detection, finding middle
- **Monotonic Stack**: Next greater/smaller element
- **Level Order Traversal**: BFS on trees
- **Tree Recursion**: Divide and conquer on trees

## Common Patterns

### Linked List Patterns
- Two pointers (fast/slow)
- Dummy head node
- Runner technique

### Stack Patterns
- Monotonic stack
- Valid parentheses
- Expression evaluation

### Tree Patterns
- DFS (preorder, inorder, postorder)
- BFS (level order)
- Divide and conquer

### Heap Patterns
- Top/Bottom K elements
- K-way merge
- Running median
